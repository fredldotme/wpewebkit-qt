# WPE WebKit Qt

WPE WebKit QML plugin

[Introducing WPEQt, a WPE API for Qt5 ](https://base-art.net/Articles/introducing-wpeqt-a-wpe-api-for-qt5/)

## Building

**Qt 6**
```
mkdir build && cd build
cmake ..
make
```

**Qt 5**

```
mkdir build && cd build
cmake -DUSE_QT6=OFF ..
make
```
